## 目的
確保在系統中只有一個實例被創建，並提供一個全局訪問點來訪問該實例。

---
<br>

## 使用時機
當系統需要確保某個類別只能創建一個實例時，或當該實例需要被全局訪問時，可以使用Singleton設計模式。
